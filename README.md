# Calculator

##Description

    *model a calculator which accepts add, subtract, multiply, divide, cancel, 
    absolute, raise to the power of two, square, negate, cuberoot, cube, repeat (n-operation) and 
    exit as commands to do basic mathematics*

##Dependencies

    1.Any OS
    1.JDK 1.8
    1.Apache Ant

##Bulding

    *ant build

##Run automated Test

    *ant runjUnit
